# Todoist

A task management application with a Spring Boot API backend and Flutter mobile frontend.

## Project Structure

- **api/** - Java/Spring Boot REST API
- **app/** - Flutter mobile app (Android & iOS)

## Getting Started

### Backend
```bash
cd api
mvn spring-boot:run
```

### Frontend
```bash
cd app
flutter run
```

## Docker

```bash
docker-compose up
```
